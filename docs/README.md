# codPass Documentation

Documentation for **codPass**, a community fork of
[sysPass](https://github.com/nuxsmin/sysPass).

## Contents

- **[INSTALL.md](INSTALL.md)** — requirements and installation (manual setup
  and a Docker pointer).
- **[CONFIGURATION.md](CONFIGURATION.md)** — where configuration lives
  (`app/config/config.xml`) and the main configurable areas: authentication,
  mail, backup, and encryption.

## A note on upstream documentation

Historical sysPass documentation and FAQ lived at `https://doc.syspass.org`.
Upstream has had no release since 3.2.11 (July 2022), and that documentation
site may now be unavailable. The pages in this folder are the maintained
reference for codPass and are based on this repository.
